# 八字玄学求欧气阵法 (SillyTavern Extension)

一个为 SillyTavern (酒馆) 编写的玄学抽卡/吃谷预测插件。
AI 将自动检索您许愿的角色/谷子背景元素，并结合您的生辰八字排出专属结印阵法。

## Features (特性)
* **无缝调用酒馆 API**：直接使用酒馆当前连接的大模型进行推演，无需额外配置。
* **支持独立第二 API**：提供自定义 API 接口设置，可单独指定支持联网搜索的模型。
* **智能游戏知识库注入**：检测到特定游戏/角色名时，自动向 AI 注入角色资料（生日、五行等），防止 AI 幻觉。
* **全国省市区级联加载**：自动拉取最新的行政区划数据。
